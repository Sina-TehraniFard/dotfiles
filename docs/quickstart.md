# クイックスタートガイド

dotvim を使い始めるための最短ガイドです。

---

## インストール

```bash
# リポジトリをクローン
git clone https://github.com/Sina-TehraniFard/dotvim.git
cd dotvim

# インストールスクリプトを実行
chmod +x install.sh
./install.sh
```

## 初回起動

```bash
nvim
```

初回起動時にプラグインが自動でインストールされます（少々時間がかかります）。

## 最低限覚えるキー操作

### ファイル操作

| キー | 機能 |
|------|------|
| `Space + e` | ファイルツリーを開く/閉じる |
| `Space + ff` | ファイルを検索して開く |
| `Space + fg` | テキストをプロジェクト全体から検索 |
| `Ctrl + s` | 保存 |

### コード編集

| キー | 機能 |
|------|------|
| `gd` | 定義にジャンプ |
| `gr` | 参照一覧を表示 |
| `K` | ドキュメント/型情報を表示 |
| `Space + rn` | 名前を変更（リネーム） |
| `gcc` | 行をコメントアウト/解除 |

### 移動

| キー | 機能 |
|------|------|
| `Ctrl + h/j/k/l` | ウィンドウ間を移動 |
| `Shift + h` | 前のバッファ（タブ） |
| `Shift + l` | 次のバッファ（タブ） |

### Git

| キー | 機能 |
|------|------|
| `Space + gg` | LazyGit を開く |
| `Space + hb` | 行のblame（誰が書いたか）を表示 |

## LSPサーバーのインストール

Neovim を起動して以下のコマンドを実行：

```
:Mason
```

Mason の画面で言語サーバーをインストールできます。
（主要なサーバーは自動インストールされます）

## 設定の確認

```
:checkhealth
```

問題がないか確認できます。
